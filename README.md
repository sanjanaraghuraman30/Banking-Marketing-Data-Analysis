# Banking Marketing Data Analysis

## Project Overview
This project performs Exploratory Data Analysis (EDA) on a banking marketing dataset to understand customer behavior and identify factors influencing term deposit subscriptions.

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## Dataset
The dataset contains over 41,000 customer records with demographic, campaign, and economic attributes.

## Analysis Performed
- Data loading and inspection
- Missing value analysis
- Data type verification
- Statistical summary generation
- Customer subscription distribution analysis
- Correlation matrix visualization

## Key Insights
- Most customers did not subscribe to a term deposit.
- Strong correlations were observed among certain economic indicators.
- Data visualization helped identify customer behavior patterns and trends.
